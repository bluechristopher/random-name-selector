# Random Student Selector

A responsive web application for randomly selecting students from predefined classes or custom lists with an elegant dark theme and interactive animations.

## Features

- **Dual Mode Selection**
  - Predefined Classes (A, B, C, D) with 25 Singaporean names each
  - Custom List mode for adding your own names

- **Smart Selection Logic**
  - Keeps track of recently selected students
  - Ensures fair distribution by avoiding repeated selections
  - Automatically resets history once all students have been selected

- **Visual Effects**
  - macOS-style rotating spinner animation
  - Smooth spinning animation with random name cycling
  - Winner celebration with pop-and-bounce animation
  - Confetti explosion on selection
  - Aurora glow effect around winner

- **Professional Design**
  - Dark theme with gradient backgrounds
  - Dynamic dropdown colors that change per class:
    - Dark green gradient
    - Dark blue gradient
    - Dark red gradient
    - Dark teal gradient
    - Dark gray gradient
  - Responsive layout for mobile and desktop
  - Light gray app background for subtle elegance

- **Additional Features**
  - Live system clock (12-hour format) in footer
  - Copyright notice with current year
  - Text wrapping for long names
  - Expandable custom list input
  - Student count display

## How to Use

1. **Select a Class**: Choose from Class A, B, C, or D from the dropdown
2. **Spin**: Click the "Spin" button or click on the result box to randomly select a student
3. **Custom List**:
   - Click "Switch to Custom List" button
   - Click the student count button to expand the input area
   - Enter names (one per line)
   - Click Spin to select

## Installation

Simply open `Random Name Generator.html` in any modern web browser.
